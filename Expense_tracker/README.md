EXPENSE TRACKER 

A simple Python-based command-line expense tracker that helps you monitor monthly spending across categories like utilities, rent, groceries, leisure, and investments. It calculates total outflow, savings, and spending percentages, rewarding good saving habits with automatic investment boosts.

Features:

- Track expenses by category

- Calculate total spending and savings

- Display spending distribution as percentages

- Incentivize saving: adds $100 to investments if savings exceed $300

- Simple console-based interface
