# Word Frequency Analyser

A lightweight Python script that analyzes a user-inputted sentence or paragraph to compute key statistics including word frequencies, the most common word, total word count, and the number of unique words.

## Features:
 **Calculates word frequencies** – Counts how many times each word appears in the text.
 **Identifies the most common word** – Displays the word with the highest frequency.
 **Displays total word count** – Counts the total number of words in the input.
 **Counts unique words** – Shows the number of distinct words in the input.
 **Simple command-line input** – Takes a sentence or paragraph as input from the user.
