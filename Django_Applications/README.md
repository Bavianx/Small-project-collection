# Django Learning Projects

A collection of Django web applications built as part of a 
structured software engineering development programme
transitioning from Python scripting and OOP into full stack 
web development.

## Purpose
This repository documents the learning progression from 
procedural Python and object-oriented programming into 
Django web framework development, database integration, 
REST APIs and full stack application architecture.

## Current Projects

### Book Catalogue
A Django web application demonstrating core Django concepts
models, database integration via ORM, and the Django admin 
interface. Built as a familiar first project using previously 
learned library system logic in a new framework context.

## Planned Projects

### Contact Manager (Django)
Rebuild of the existing CLI Contact Manager as a full Django 
web application with user authentication and database 
persistence.

### Expense Tracker (Django)
Rebuild of the existing CLI Expense Tracker with web 
interface, multi-currency support and data visualisations.

### Doomsday Clock
A full stack Django application tracking global geopolitical 
tensions using live news APIs, economic indicators and 
threat vector scoring. Features real time data ingestion, 
weighted risk scoring algorithm, Plotly visualisations and 
PostgreSQL database. The centrepiece portfolio project 
demonstrating full stack capability.
