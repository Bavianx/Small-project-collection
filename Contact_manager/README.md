# Contact Catalogue 
A command-line contact management system built in Python that stores worker information with full data persistence.

# Features

- Add Workers – Store worker name, email, mobile and job role
- Edit Workers – Update any field with confirmation prompts
- Remove Workers – Delete workers with confirmation
- View All Workers – Display full catalogue at a glance
- Data Persistence – Automatically saves to JSON so data survives restarts
- Backup System – Creates automatic backups on every save and load
- Input Validation – Handles empty inputs and invalid characters
- Error Handling – Gracefully handles corrupted files and permission errors
