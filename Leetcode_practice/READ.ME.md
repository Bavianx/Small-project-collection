
LEETCODE PRACTICE & ALGORITHM LEARNING

A Python-based collection of learning implementations of LeetCode problems and LeetCode-style algorithm exercises.
This repository focuses on building foundational problem-solving skills using arrays, strings, hash maps, and sets while documenting the learning process.

The solutions are written for practice and understanding, not as final optimized submission




Official LeetCode Problems Included

- Two Sum (LeetCode #1)

- Best Time to Buy and Sell Stock (LeetCode #121)

- Contains Duplicate (LeetCode #217)

- Valid Anagram (LeetCode #242)

- Valid Palindrome (LeetCode #125)

- Palindrome Number (LeetCode #9)

- First Unique Character in a String (LeetCode #387)
